## Introduction
This project is a web server written in Java. It parses GET, POST, PUT, DELETE requests, can handle static resources, and supports HTTP long connections.
## Features

* Servlet Container
* Dispatcher
* Request
* Response
* GET/POST
* web.xml Parse
* NIO
* Static Resources
* Cookies
* log4j
